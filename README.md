# Portfolio-simple-website

This is a basic personal portfolio website created using **HTML** and **CSS**.  
The project includes a simple and clean design with two main sections:

- **Home Page** – Introduction and welcome section  
- **Contact Page** – Basic contact information/form  

## Features
- Responsive and simple layout
- Clean user interface
- Beginner-friendly project structure
- Built only with HTML and CSS

## Technologies Used
- HTML5
- CSS3

## Purpose
This project was created to practice front-end web development basics and improve understanding of website structure and styling using pure HTML and CSS.
